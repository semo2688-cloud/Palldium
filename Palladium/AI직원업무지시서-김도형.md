## 2. 만들 AI 직원의 역할

**이름**: 중고 에어컨 데이터 수집 및 정제 AI 직원

**하는 일**: 당근마켓 또는 중고나라의 중고 에어컨 상품 링크를 받아 다음 11개 항목을 추출하고, 중복을 제거한 후 스프레드시트로 정리

| 항목 | 설명 |
|------|------|
| 플랫폼 | 당근마켓, 중고나라 등 |
| 브랜드 | LG, 삼성, 대우 등 |
| 모델명 | 정확한 제품 모델 |
| 형태 | 벽걸이형, 스탠드형 등 |
| 연식 | 제조 연도 |
| 가격 | 판매 가격 |
| 상태 | 미사용, 거의새것, 좋음, 보통, 나쁨 등 |
| 이미지 URL | 상품 사진 링크 |
| 지역 | 판매 지역 |
| 출처 URL | 원본 상품 페이지 링크 |
| 비고 | 추가 정보 |

## 3. 오늘 MVP 범위

**수업 당일 목표**: 당근마켓 또는 중고나라에서 100~300개 샘플 데이터를 수집하고, 중복을 제거한 깨끗한 데이터셋을 Notion 또는 엑셀로 완성

**처리 흐름**

```
1단계: 입력 준비
  ↓
2단계: AI가 링크에서 11개 항목 추출
  ↓
3단계: 중복 데이터 검토 및 제거
  ↓
4단계: 최종 데이터셋을 스프레드시트로 저장
```

**입력값**
- 당근마켓/중고나라 중고 에어컨 상품 링크 목록 (100~300개)
- 11개 항목 추출 지시문 (위 표 참고)

**처리 도구**
- Claude 또는 ChatGPT (링크 읽기 및 정보 추출)
- Notion 또는 엑셀 (데이터 저장)

**출력값**
- 정제된 중고 에어컨 데이터셋 (CSV, 엑셀, 또는 Notion 테이블)
- 각 행: 1개 상품 = 11개 항목 완성

## 4. 신청자가 준비해야 할 것

**필수 입력 자료**

1. **링크 목록** (100~300개)
   - 당근마켓 또는 중고나라의 중고 에어컨 상품 링크
   - 형식: 텍스트 파일, 스프레드시트, 또는 목록 형태
   - 예시:
     ```
     https://www.daangn.com/articles/1234567890
     https://www.daangn.com/articles/1234567891
     ...
     ```

2. **추출 지시문** (AI에게 전달할 프롬프트)
   - 위 표의 11개 항목을 어떻게 추출할지 명확히 기술
   - 예: "각 링크에서 다음 정보를 추출하세요: 플랫폼(당근마켓/중고나라), 브랜드, 모델명..."

**선택 사항**
- Notion 또는 엑셀 템플릿 (미리 준비하면 더 빠름)

## 5. 개발 시 주의사항

**플랫폼 약관 확인**
- 당근마켓, 중고나라 등 각 플랫폼의 약관에서 자동 수집 제한 여부 확인
- 공식 API 제공 여부 먼저 확인
- 약관 위반 시 서비스 이용 제한 가능성 있음

**API 키 보안**
- Claude API 키는 코드에 직접 입력하지 말 것
- 환경변수(.env) 파일로 관리
- GitHub 등 공개 저장소에 .env 파일 업로드 금지

**개인정보 처리**
- 수집된 데이터에 개인정보(판매자 연락처, 주소 등) 포함 시 별도 처리 필요
- 테스트 단계에서는 샘플 데이터로 진행

## 8. 개발 진행 순서

**Step 1: 입력 인터페이스 구성**
- 링크 목록 입력 영역 (텍스트 또는 파일 업로드)
- 추출 지시문 입력 영역
- 실행 버튼

**Step 2: AI 처리 로직**
- Claude API 또는 ChatGPT API 연결
- 각 링크에서 11개 항목 추출하는 프롬프트 작성
- 응답 파싱 및 정제

**Step 3: 중복 제거 로직**
- 추출된 데이터에서 중복 판단 기준 정의 (예: 출처 URL 동일, 브랜드+모델명+가격 동일)
- 중복 제거 알고리즘 구현

**Step 4: 출력 및 저장**
- 정제된 데이터를 CSV 또는 JSON 형식으로 생성
- Notion API 또는 엑셀 파일로 직접 저장 (선택)
- 사용자가 다운로드할 수 있게 제공

**Step 5: 테스트 및 마무리**
- 샘플 링크 5~10개로 전체 흐름 테스트
- 오류 처리 및 사용자 안내 메시지 추가
